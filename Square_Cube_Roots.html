SECTION 5

Let's Compare! 📊

Square roots vs Cube roots — side by side

√ Square Root
Symbol:√ (radical sign, no small number)
Meaning:Reverse of squaring (×2 times)
Example:√49 = 7 because 7² = 49
Memory:Square = flat 2D grid ◻️
³√ Cube Root
Symbol:³√ (small 3 before radical)
Meaning:Reverse of cubing (×3 times)
Example:³√27 = 3 because 3³ = 27
Memory:Cube = solid 3D block 🎲

Not all numbers are perfect squares — but we can estimate!

📚 Key Vocabulary

🔷Square Number

A number obtained by multiplying an integer by itself.

Example: 4 × 4 = 16 → 16 is a square number
√Square Root

The number that when multiplied by itself gives the original number.

Example: √25 = 5 because 5² = 25
🎲Cube Number

A number obtained by multiplying an integer by itself three times.

Example: 3 × 3 × 3 = 27 → 27 is a cube number
³√Cube Root

The number that when cubed gives the original number.

Example: ³√64 = 4 because 4³ = 64
📐Perfect Square

A number whose square root is a whole number (integer).

Examples: 1, 4, 9, 16, 25, 36, 49, 64, 81, 100
🧊Perfect Cube

A number whose cube root is a whole number (integer).

Examples: 1, 8, 27, 64, 125, 216, 343
📏Consecutive Integers

Whole numbers that follow one after another in order.

Example: 8 and 9 are consecutive integers
🏷️Index / Exponent

The small number written above a base showing how many times to multiply.

In 5², the index is 2 (meaning 5 × 5)
